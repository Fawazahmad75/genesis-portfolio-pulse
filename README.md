# Genesis Portfolio Pulse

An AI-powered intelligence dashboard for tracking activity across Genesis incubator's portfolio companies.

**Status:** In active development for the Genesis Builder-in-Residence application (May 2026).

Full README with architecture, decisions, and live demo coming after frontend deploy.

## Stack
- FastAPI + SQLite + ChromaDB
- Firecrawl for site-only ingestion
- Groq (Llama 3.3 70B) for classification, digest, trends
- sentence-transformers for embeddings
- Lovable (frontend, in progress)
